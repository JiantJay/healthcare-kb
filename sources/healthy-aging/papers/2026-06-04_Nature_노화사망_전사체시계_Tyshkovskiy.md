## 핵심 요약 (두괄식)

**노화를 "시간의 흐름"이 아니라 "측정 가능한 분자적 상태"로 재정의한 연구로, 포유류 4종(생쥐·쥐·마카크·인간) 25개 이상 조직의 전사체 11,000개 이상을 통합해 노화와 사망위험의 공통(보편적) 분자 신호를 규명하고, 단순 나이 예측을 넘어 실제 건강 악화·사망위험을 더 잘 반영하는 "사망 전사체 시계(mortality transcriptomic clock)"를 구축했다.** 핵심은 노화가 단일 현상이 아니라 염증·인터페론·미토콘드리아·염색질·세포외기질 등 **생물학적 모듈별로 다르게 진행**된다는 점을 모듈별 시계로 정량화한 것이다.

주요 발견:
- **두 종류의 시계**: chronological age(달력 나이) 예측 시계 + expected mortality(예상 사망위험) 시계. mortality clock은 수명 단축 모델에서 transcriptomic age 증가, 칼로리 제한·수명연장 개입에서 감소하는 방향성을 잘 포착
- **보편 바이오마커**: CDKN1A(p21), LGALS3(galectin-3), GPNMB, Eda2r, Vsig4 등이 종·세포 유형을 가로질러 반복 등장. CDKN1A·LGALS3는 UK Biobank에서 단백질 수준도 사망·다중질환과 연관
- **손상 모델 재현**: 염증, 복제 노화(senescence), 대사 억제, γ선 조사 등 손상 축적 모델에서 사망 관련 특징이 재현되었고, 세포 불멸화·리프로그래밍·이종 개체결합(parabiosis)·초기 배아발생에서 약화·역전
- **모듈별 차등 노화**: 만성질환은 주로 염증·인터페론 모듈 노화를 가속 / 칼로리 제한·Klotho(Kl) 결핍은 미토콘드리아·대사 모듈을 표적
- **다종 일반화**: 인간 혈액에서 전사체 시계와 DNA 메틸화 시계의 나이 가속이 상관(특히 염색질 모듈 시계에서 최강) → 분자 노화 양식 간 기전적 연결
- **장수 유전자**: Igf1(노화 감소·수명 단축 연관), Ddost(AGE 처리)와 대비되는 pro-longevity 유전자 Fmo3(mTOR·염증 억제), Nmrk1(NAD+ 생합성)

## 주제별 정리

### 1. 배경 — 기존 시계의 한계
- DNA 메틸화 시계는 생물학적 나이 측정에 강력하나, "어떤 유전자·경로가 노화·사망위험을 실제로 움직이는지" 해석에 한계
- 전사체 모델은 유전자가 기능적으로 주석되어 **해석 가능성**이 높고, scRNA-seq/snRNA-seq로 세포 유형별 분자 나이 정량 가능

### 2. 데이터·모델 구축
- 설치류: UM-HET3 마우스에 20종 ITP(노화중재시험) 개입 + 공개 전사체·생존 데이터 통합 → 4,539 설치류 샘플(26개 조직). Gompertz 모델로 개체별 expected log hazard(예상 사망위험) 산출
- 다종 확장: 영장류 6,626 샘플(마카크 2,623 / 인간 4,003: 뇌·골격근·피부·혈액) 추가 → "universal" 다종·다조직 상대 시계
- 성능: 종 내 Pearson r ≥ 0.94(chronological), r ≥ 0.91(mortality). 전체 r = 0.952로 범포유류 DNA 메틸화 시계(r = 0.953)와 동등. 미학습 종에서도 0.66 ≤ r ≤ 0.84로 일반화

### 3. 보편 신호와 경로
- 종·조직 공통 상향: Gpnmb, Vsig4, Cdkn1a, Eda2r / 공통 하향: Nrep, Col1a1, Col3a1
- 경로: 인터페론·인터루킨·p53 신호, 보체/응고, 염증·스트레스 반응 = 노화·사망과 양의 연관, 수명과 음의 연관 / 산화적 인산화·미토콘드리아 번역 = 반대 패턴. 지방산·xenobiotic 대사도 사망과 음의 연관

### 4. 모듈별 시계 (핵심 기여)
- 노화·사망 신호를 공동조절 모듈(염증, 인터페론, 미토콘드리아 기능, 염색질 수정, 세포외기질)로 분해 → 모듈별 시계 개발
- 개입의 경로 특이적 효과: 만성질환 → 염증 모듈 노화 가속 / 칼로리 제한·Klotho 결핍 → 미토콘드리아·대사 모듈 표적

### 5. 한계와 임상 적용 과제 (저자·정리 관점)
- 전사체는 조직 상태·환경 변화에 민감 → 임상 적용에는 **혈액 기반 검증, 장기 추적 연구, 단백질 바이오마커와의 통합**이 추가로 필요
- 의의: 건강수명 연구, 항노화 개입 평가, 만성질환 위험 예측의 기반 프레임워크
