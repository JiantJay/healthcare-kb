# Wiki Index — 위키 레이어 인덱스

> 이 폴더는 LLM Wiki의 "위키 레이어"임. `sources/`(raw, 불변 원본)를 ingest하여 생성한 개념·엔티티·분석 페이지를 보관. 모든 페이지는 1페이지 = 1개념, 상단 한 줄 요약 + 태그, 본문 내 `[[백링크]]` 연결, 하단 출처(sources/ 경로) 표기.
